# OSINT Methodology

## Process
1. **Scope** — who/what/where/when
2. **Gather** — web search, social media, public records
3. **Verify** — cross-reference, check dates, check sources
4. **Filter** — keep signal, discard noise
5. **Structure** — standard brief format
6. **Flag** — confidence level + knowledge gaps

## Confidence Levels
| Level | Criteria |
|-------|----------|
| **High** | Multiple independent reliable sources agree. Official records exist. |
| **Medium** | 2+ sources but one may be derivative. Unverifiable official source. |
| **Low** | Single source. Uncorroborated claim. Anonymous or unreliable. |

## Common Pitfalls
- Confirmation bias — don't cherry-pick
- Source chain contamination — many citing one = still one source
- Outdated info — check timestamps
- Misattribution — verify claim matches source
